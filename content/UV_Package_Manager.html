UV: The Lightning-Fast Python Package Manager

← Back to Home

If you've ever felt frustrated by the slow performance of pip or the complexity of managing Python environments, UV is here to change the game. Built in Rust by the team at Astral, UV is a blazingly fast Python package and project manager that's designed to be a drop-in replacement for pip, pip-tools, and even poetry.

What is UV?

UV is an extremely fast Python package installer and resolver. It's written in Rust, which gives it performance that's 10-100x faster than traditional tools like pip. But speed isn't its only advantage — UV also handles virtual environments, dependency resolution, and project management all in one unified tool.

Why Choose UV?

  • Speed: Up to 100x faster than pip for package installation
  • Modern Dependency Resolution: Smart conflict resolution and faster lockfile generation
  • All-in-One Tool: Replaces pip, venv, pip-tools, and poetry with a single command
  • Cross-Platform: Works seamlessly on Windows, macOS, and Linux
  • Drop-in Replacement: Compatible with existing requirements.txt and pyproject.toml files

Installation

Installing UV is straightforward. On macOS and Linux:

curl -LsSf https://astral.sh/uv/install.sh | sh

On Windows (PowerShell):

powershell -c "irm https://astral.sh/uv/install.ps1 | iex"

Or using pip:

pip install uv

Basic Commands

Creating a Virtual Environment

UV makes creating virtual environments incredibly simple:

uv venv

This creates a .venv directory in your current folder. To activate it:

# On macOS/Linux
source .venv/bin/activate

# On Windows
.venv\Scripts\activate
Installing Packages

Install packages just like you would with pip, but much faster:

# Install a single package
uv pip install requests

# Install from requirements.txt
uv pip install -r requirements.txt

# Install specific version
uv pip install "numpy==1.24.0"
Syncing Dependencies

UV can sync your environment to match a requirements file exactly:

uv pip sync requirements.txt

This ensures your environment has exactly what's specified — no more, no less.

Compiling Dependencies

Generate a locked requirements file with resolved dependencies:

uv pip compile requirements.in -o requirements.txt

Project Management with UV

UV also supports full project workflows similar to poetry:

Initialize a New Project
uv init my-project
cd my-project
Add Dependencies
uv add requests pandas numpy
Run Scripts in Project Environment
uv run python main.py

Advanced Features

Python Version Management

UV can even manage Python versions for you:

# Install a specific Python version
uv python install 3.11

# List available Python versions
uv python list

# Use a specific Python version
uv venv --python 3.11
Cache Management

UV caches packages for faster reinstalls. Manage the cache with:

# Show cache size
uv cache dir

# Clear the cache
uv cache clean
Lock Files

Generate reproducible builds with lock files:

uv lock

Migration from pip

Switching from pip to UV is straightforward. Here's a quick migration guide:

pip command UV equivalent
python -m venv .venv uv venv
pip install package uv pip install package
pip install -r requirements.txt uv pip install -r requirements.txt
pip freeze > requirements.txt uv pip freeze > requirements.txt
pip list uv pip list

Best Practices

  • Use lock files: Always commit uv.lock to ensure reproducible builds
  • Leverage the cache: UV's cache makes repeated installs instant
  • Use uv sync: For exact dependency matching in production
  • Pin Python versions: Use --python flag to ensure consistency across teams
  • Combine with CI/CD: UV's speed makes it perfect for continuous integration pipelines
